# 🧮 Calculator App (Python Tkinter)

A simple and modern calculator application built using **Python** and **Tkinter**.
It provides basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔄 Clear (C) button
* ⌫ Backspace support
* 🎨 Modern dark-themed UI

---

## 🛠️ Technologies Used

* Python 🐍
* Tkinter (GUI Library)

---

calculator/

## ▶️ How to Run

1. Clone the repository:

```
git clone [https://github.com/AnishKulal08/calculator.git](https://github.com/Anishkulal08/Pyhton_calculator-GUI-.git)
```

2. Navigate to the folder:

```
cd calculator
```

3. Run the program:

```
python calculator.py
```
